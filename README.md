# Бинарный поиск и Двоичные деревья в ядре Linux

Это учебный проект, демонстрирующий работу со структурами данных в пространстве ядра Linux (kernel space).

Проект состоит из двух модулей ядра:
1.  **`ex_bin_tree.ko`**: Реализует Красно-Черное дерево (Red-Black Tree) для хранения и управления пользователями.
2.  **`ex_bin_search.ko`**: Демонстрирует классический бинарный поиск по отсортированному массиву структур.

## Требования

Для сборки и запуска модулей вам понадобится:
*   **ОС**: Любой дистрибутив Linux (проверено на Ubuntu).
*   **Компилятор**: Версия GCC должна совпадать с версией, использованной для сборки вашего ядра.
*   **Заголовки ядра**: Установленный пакет `linux-headers-$(uname -r)`.

Установить заголовки на Ubuntu/Debian можно командой:

sudo apt-get update && sudo apt-get install build-essential linux-headers-$(uname -r)

## Сборка и запуск

В корневой директории проекта находится `Makefile`, который автоматизирует процесс.

### 1. Сборка модулей
Откройте терминал в папке проекта и выполните:

make

*Если вы видите предупреждение о версии GCC, но сборка проходит успешно — это нормально.*

### 2. Загрузка модулей в ядро

sudo make run

Эта команда соберет модули (если они еще не собраны), загрузит их (`insmod`) и выведет последние строки системного журнала (`dmesg`), где вы увидите результат работы.

### 3. Выгрузка модулей

sudo make remove



### 4. Очистка проекта (удаление файлов сборки)

make clean
